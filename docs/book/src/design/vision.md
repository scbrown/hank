# Vision

The north-star vision document — *Bobbin × Hank × Quipu: A Governed,
Multi-Signal, Multi-Tenant Code Intelligence Layer* — lives at the repository
root:

- **[`docs/vision.md`](https://github.com/scbrown/hank/blob/main/docs/vision.md)**

The thesis: Bobbin answers *"what code is relevant?"* with embedding similarity
and historical co-change. Neither knows the actual structure or semantics of the
code. Hank extracts and serves live per-tenant structure; Quipu governs and
versions the committed record; Bobbin fuses everything — making autonomous
agents better-informed, safely-bounded, and correct even when a whole team is
editing at once.
